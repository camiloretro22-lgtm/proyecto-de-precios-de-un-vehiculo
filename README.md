# proyecto-de-precios-de-un-vehiculo

El propósito principal es demostrar habilidades en ingeniería de software, incluyendo:
- Configuración de entornos virtuales de Python.
- Desarrollo de una aplicación web con **Streamlit**.
- Creación de gráficos interactivos con **Plotly Express**.
- Despliegue continuo (CI/CD) a través de **GitHub**
